# Game Shop
## Preconditions
https://nodejs.org/en

## Commands
- To start the front-end (in "/GameShop/frontend/")
  ```
  & "C:\Program Files\nodejs\npm.cmd" start
  ```
  or
  ```
  npm start
  ```
  
- To build the front-end (in "/GameShop/frontend/")
  ```
  & "C:\Program Files\nodejs\npm.cmd" run build
  ```
  or
  ```
  npm run build
  ```
  
- Copy and paste everything (after building the front end) from
  ```
  Gameshop/frontend/build
  ```
  to
  ```
  GameShop\src\main\resources\static
  ```
  
- Run (in "GameShop/")
  ```
  ./gradlew run
  ```

## Database
